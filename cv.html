Compétences Techniques

Formation Développeur Web à KADEA ACADEMIE

Langages :

HTML5 :

Maîtrise des balises sémantiques et création de pages web accessibles.

CSS3 :

Utilisation avancée de la mise en page (Flexbox, Grid), gestion des couleurs, typographies et animations CSS.

JavaScript :

Manipulation du DOM, gestion des événements utilisateurs, création de scripts dynamiques pour améliorer l’interactivité des pages

Outils de développement :

Visual Studio Code :

Utilisation de l’éditeur pour écrire, déboguer et organiser le code de manière efficace.

Git & GiyHub :

Gestion des versions de projets via Git, utilisation de GitHub pour le partage de code et la collaboration.

Chrome DevTools :

Analyse des performances des pages web, identification et correction des erreurs via les outils intégrés du navigateur.

Méthodologies et Concepts :

Responsive Design :

Création de sites web qui s'adaptent à différentes tailles d'écran (mobile, tablette, bureau).

Débogage :

Recherche et correction des bugs dans le code JavaScript et les feuilles de style CSS.

SEO de base :

Optimisation des pages web pour les moteurs de recherche via des pratiques HTML adaptées.
